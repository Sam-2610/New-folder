# 💰 Expense Tracker

A stylish and responsive **Expense Tracker Web App** built using HTML, CSS, and JavaScript. It helps you manage your income and expenses, calculate balances automatically, and store all data in the browser using **localStorage**.

---

## 📌 Features

- 📥 Add income and expense transactions
- 💸 Calculates total balance, income, and expenses
- 🗑️ Delete transactions easily
- 💾 Data is saved locally using **localStorage**
- 📱 Fully responsive design
- 🎨 Smooth animations and modern UI



## 🛠️ Built With

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
- **localStorage API**

---

## 🧪 How to Use

1. Clone the repository or download the `.zip` file.
2. Open `index.html` in your browser.
3. Add your transactions by entering:
   - A **description**
   - An **amount** (`positive` for income, `negative` for expense)
4. Your balance updates in real time.
5. Transactions are saved in localStorage and will persist across sessions.
